# 编辑器内第一轮操作

## 目标

让项目第一次在 Cocos Creator 里进入“可开始搭场景”的状态。

## 先做这 4 步

1. 在 Dashboard 项目列表里，把 `wisdom` 绑定到 `3.8.8`
2. 打开项目
3. 等编辑器第一次导入资源
4. 确认项目根目录出现 `library/`、`temp/`、`settings/`、`local/`

## 如果项目仍显示 `缺失编辑器`

说明只是 Dashboard 还没给这个项目指定版本，不是项目坏了。

直接在项目这一行的编辑器版本列里：

- 选择 `3.8.8`
- 再点击打开

## 打开后第一批节点

建议先手动建这些节点：

```text
Canvas
PersistentRoot
World
  Player
    AttackAnchor
  EchoRoot
  RoomRoot
  TriggerRoot
  EnemyRoot
```

## 第一批挂载

### PersistentRoot

- `GameManager`
- `SceneLoader`
- `SaveSystem`
- `KeyboardInputDriver`

说明：

- `KeyboardInputDriver.player` 指向 `Player`
- `GameManager.playerRoot` 指向 `Player`

### Player

- `PlayerController`
- `HealthComponent`

### EchoRoot

- `EchoManager`

### RoomRoot

- `RoomResetController`

### TriggerRoot

先准备 3 个测试节点：

- `Checkpoint`
- `Plate`
- `Trap`

分别挂：

- `CheckpointMarker`
- `PressurePlateSwitch`
- `ProjectileTrap`

## 桌面调试按键

- `WASD / 方向键`：移动
- `J`：攻击
- `K`：放置当前回响

## 你做完这一轮后，项目就进入下一个阶段

到这一步，我们就可以继续搭：

- 木箱 prefab
- 压板机关
- 子弹 prefab
- 第一条 2 分钟机制流程
