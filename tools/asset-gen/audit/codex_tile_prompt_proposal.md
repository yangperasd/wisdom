# Tile Prompt Rework Proposal

## Failure analysis (per candidate)
### outdoor_wall_standard_v00 (3.93)
Visual: Busy cream-grey wall with mossy blotches and several medium-large block shapes; the right-mid green cluster and darker stone patches read like repeatable stamps when tiled. Palette skews olive-grey instead of sunny sandstone.
R3 defects named: left/right edge mismatch and top/bottom mismatch, visible seam grid in 3x3 tiling, gradient bleed in stone fills, soft mortar, speckle noise, cool/desaturated stone, dark olive greens, weak warmth. Consistency-breaker also flagged muddy/photo-derived noise, though its peer set looks mismatched.
Root cause in current prompt: "large blocks," "ivy accents," and "soft shadows" invite macro features and realistic shading; the prompt never says no focal cluster, balanced density, or low-contrast repeating structure.

### outdoor_wall_broken_v00 (3.93)
Visual: Broken wall reads darker and busier, with rubble-like dark pockets and moss clumps that create obvious repeat anchors. It feels closer to a ruin/dungeon wall than a bright outdoor Zelda wall.
R3 defects named: all four edges mismatch, visible seams on both axes, gradient bleed in stone areas, moss color bleed, noisy bottom-right rubble patch, cold grey stone, dark mossy greens, shadow patches drifting toward dungeon tone.
Root cause in current prompt: "crumbling," "missing bricks," "rubble," and "ivy draped over" push the model toward one-off damage clusters and dark cavities instead of evenly distributed chip damage that can wrap.

### outdoor_path_cobble_v00 (4.10)
Visual: Dark charcoal cobbles with teal-olive moss and several oval shadow blobs. The stones feel sparse but high-contrast, so the repeated tile reads like stamped spots on a dark grid.
R3 defects named: visible seam lines in 3x3 tiling, left/right and top/bottom mismatch, blurry painted cobbles, gradient bleed, melting edges into grout, green bleed, speckle in channels, palette too dark, cold grey stone, grim dungeon-floor mood with no warm tans.
Root cause in current prompt: "irregular pattern" plus vague "moss edges" gives the model permission to invent large dark gaps and uneven stone clusters; there is no instruction for warm outdoor stone, uniform stone scale, or no visible repeat hotspot.

### outdoor_ground_green_v00 (4.92) - relatively better
What's working: seam geometry is much better, the tile reads bright and sunny, and detail stays mostly micro-scale instead of forming one big object. R3 still noted too many green tones and a few stray blue pixels, so it needs tighter palette control, not a full concept change.

### outdoor_ground_ruins_v00 (4.87) - relatively better
What's working: crisp pixel blocks, readable small debris, and cleaner hard edges than the wall/path failures. The main misses are still edge asymmetry and a cool olive-grey palette; structurally it is closer to usable once warmed up and made more evenly distributed.

## Common failure patterns across all 7
1. "Seamless" is present, but the prompts do not force wrap-safe composition: no explicit ban on focal subjects, no requirement for balanced density, and no instruction that all four edges must wrap invisibly.
2. Several prompts ask for macro motifs that are too large for a 32x32 support tile: large blocks, zigzag crack, missing bricks, rubble, ivy drape, or flower clusters. After 1024 to 32 downscale, those become repeating stamps.
3. Palette language is too loose. Stone prompts drift to cool grey/olive realism instead of warm tan sandstone; ground prompts can drift into muddy or over-noisy green.
4. Style constraints are too weak for Pixel Art XL. Without "crisp hard-stepped 2-4 tone shading" and "low visual energy," the model invents gradient bleed, airbrushed stone, and photo-like speckle.

## Testing recommendations
After regeneration, judge the 3x3 tile preview before anything else: if you can trace a grid line or spot the same blob every 32 pixels, the prompt still failed. The pass case should look quiet under a 32x32 player sprite, with warm sunny color at scene distance and only micro-variation up close. I would weight seam invisibility, absence of repeat hotspots, and warm outdoor palette above all peer-comparison feedback, since some of the current consistency checks appear contaminated by mismatched icon references.
